Print dates of a year with an offset Open on JetBrains Academy

Write a program that prints all dates of the year with a specified offset applied. 

It should read a starting date and a value of an offset (in days).

In the output, dates should be printed in ascending order with the starting date included.


Sample Input:
2017-12-19
4
Sample Output:
2017-12-19
2017-12-23
2017-12-27
2017-12-31

Sample Input:
2017-06-30
60
Sample Output:
2017-06-30
2017-08-29
2017-10-28
2017-12-27


Memory limit: 256 Mb
Time limit: 5s
